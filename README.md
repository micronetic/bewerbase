# Bewerbase

```
git clone https://github.com/micronetic/bewerbase.git
cd bewerbase
```

```
cd backend
yarn install
```

```
cd frontend
yarn install
yarn dev
```

Verwendete [Arbeitsamt API](https://jobsuche.api.bund.dev/)

https://micronetic.github.io/bewerbase
